# ML - Machine Learning

Esta pasta contém os modelos de machine learning e código relacionado ao treinamento e inferência.

## Propósito

Aqui serão implementados:
- Modelos de ML para análise preditiva
- Scripts de treinamento
- Feature engineering
- Validação e avaliação de modelos
- Notebooks de experimentação (Jupyter)
- Utilitários para processamento de dados para ML

## Estrutura Futura

```
ml/
├── models/          # Modelos treinados e salvos
├── training/        # Scripts de treinamento
├── inference/       # Código para inferência
├── features/        # Feature engineering
├── notebooks/       # Jupyter notebooks de experimentação
└── utils/           # Utilitários para ML
```
